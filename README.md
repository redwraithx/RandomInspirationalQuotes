## Random Inspirational Quotes Project

#### With over 100 quotes for each database will give enough quotes that you may never see a repeat.

##### The Quotes are combinations from using 3 data arrays that of the dataObject. dataObject.getRandomSentenceQuote() returns a random Quote using 1 entry from each of the data arrays.
##### The 3 Data Arrays within dataObject each have a theme: Identity, Action and Validation. Used once per sentence will return a quote for inspiring the reader.

#### This project was my first project to use javascript, I hope to improve on the basic Front-End in the near future and I may even add other types of random messages to this project
